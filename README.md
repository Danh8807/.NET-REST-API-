StudyAbroad API

A minimal REST API built with .NET 10 for managing study abroad student and university data.

Tech Stack


.NET 10 Minimal API
Entity Framework Core + SQLite
JWT Authentication
BCrypt password hashing
AutoMapper
Swagger UI


Features


Authentication — Register and login with JWT token
Students — Create, read, update, delete student records
Universities — Create, read, update, delete university records
Recommendations — Get university recommendations based on student GPA and SAT score
Protected Routes — All endpoints require a valid JWT token
